This document describes **the why, the who, and the how** of this project.
It is not a technical manual, but an alignment guide for users,
contributors, and maintainers.

## 2. Future Vision

<!--
The future vision describes **where the project wants to go**.
It serves as a reference to evaluate whether changes and contributions
are aligned with the desired direction.
-->

<!-- Describe the ideal state of the project in the medium/long term.
-->

* How should the project be used in 1–3 years?
- Scan automático de todos os paths de uma API e geração automática de testes de contrato
- Geração de documentação automática da API e de reports de saúde e performance da API
- ter decorator a ser usado nas rotas da API que gera automaticamente os testes de contrato
- ser referência de devtool no mundo

* What impact does it aim to generate?
- ser referência de devtool de testes de contrato na comunidade open-source

* What would “success” mean for this project?
- Ter uma comunidade engajada no projeto, 10k stars in 3 years
- financiamento para poder ter um time fulltime de suporte do projeto
